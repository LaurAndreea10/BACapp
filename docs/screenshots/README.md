# Capturi recomandate pentru BAC Space

Acest folder este pregătit pentru capturile reale folosite în README, prezentare și portofoliu.

## Capturi principale

Adaugă imaginile cu aceste nume:

1. `learning-paths.png`
   - Secțiunea **Alege traseul tău**
   - Include cele 3 carduri: Română, Uman, Real / Științe

2. `romanian-flow.png`
   - Flux complet Română
   - Ideal: tabul Lecții sau Quiz

3. `human-flow.png`
   - Flux complet Istorie + Geografie
   - Ideal: tabul Plan 7 zile sau Lecții

4. `science-flow.png`
   - Flux complet Matematică + Științe
   - Ideal: tabul Lecții sau Quiz

5. `bac-pack.png`
   - BAC Pack
   - Ideal: simulare pe profil sau raport după simulare

6. `ai-coach.png`
   - AI Coach cu sugestii grupate

7. `accessibility-panel.png`
   - Panoul ♿ deschis

8. `privacy-banner.png`
   - Bannerul simulator + confidențialitate

## Recomandări pentru capturi

- Folosește browser desktop la lățime aproximativă 1440px.
- Fă și cel puțin o captură mobilă dacă vrei să arăți responsive design.
- Evită capturile cu DevTools deschis.
- Fă hard refresh înainte de capturi.
- Dacă un banner acoperă ceva important, închide-l doar după ce ai făcut captura dedicată pentru privacy.

## Cum le inserezi în README

După ce adaugi fișierele PNG, poți activa imaginile în README cu linkuri de forma:

```md
![Alege traseul tău](docs/screenshots/learning-paths.png)
```
